# Data Pages

## Data Format
This page details the differences between the types of data available for download. The top of the page has links to download data in each form, as well as a brief description of what each includes. Farther down the page has more details about each download package. 

## Quality Flags
This page explains how to read the quality flags CORIOLIX assigns to each data value. 

## Download: High Resolution Data
This page has buttons to download High Resolution data from CORIOLIX. You can choose to download all the data from the most recent cruise or you can customize it to get specific data using the "Customize" box and filling out the options based on what you want.  

## Download: Binned Data
This page has buttons to download binned data from the most recent cruise. Data are provided as one-minute bins in CSV files. Like the "Download: High Resolution Data" page, you can download everything or just specific data using the "Customize" option. Refer to the Data Format page for information on what is included in the download using this option. 

## Download: Binning Statistics
This page has buttons to download binned data with statistics from the most recent cruise. Like the the other two data download pages, you can download everything or just specific data using the "Customize" option. Refer to the Data Format page for information on what is included in the download using this option. 

## REST API Access
This page explains how to get access to the REST API for programs like MatLab and R. 

## ERDDAP Access
This page explains how to access data through the ERDDAP server and has some example queries.
