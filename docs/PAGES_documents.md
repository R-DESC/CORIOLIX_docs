# Documents Page

The Documents page contains a list of all documents associated with your ship. This includes calibration files, images, sensor manuals, and other vendor-supplied information that has been uploaded to CORIOLIX. Documents linked to a specific sensor, such as a manual or calibration file, can also be found on the appropriate Sensor Info pages.

## Adding a New Document

To upload a new document, click the **Add New** button at the top or bottom of the document list. This will open the **Document Details and Upload** form.

### Document Details

Fill in the following fields to describe your document:

- **Category** *(required)* - the type of document
- **Title** *(required)* - a short, descriptive name for the document
- **Author** - the person or organization who created the document
- **Document Date** - the date the document was created or published
- **Version** - the version number of the document, if applicable
- **Access Level** - set to **Public** to make the document visible to all users

### Document Family

Select **one** of the following four categories to associate the document with the correct part of CORIOLIX:

- **Associated Sensor** - use this for documents like calibration files
- **Associated Sensor Model** - use this for documents like sensor user manuals and datasheets
- **Associated Cruise** - use this for documents like cruise reports and participant lists
- **Associated Vessel** - use this for documents like vessel schematics

### Document Upload

Once your details are filled in, upload your file. The maximum supported file size is **100 MB**.

## Editing and Deleting Documents

> ðŸ”’ **Editing and deleting documents requires you to be logged in.**

Each document in the list has an **Edit** button and a **Delete** button.

- **Edit** - opens the Document Details form so you can update the document's metadata or replace the file
- **Delete** - removes the document permanently. Use with caution, as this action may not be reversible
