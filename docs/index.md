## Overview

CORIOLIX (**C**ruise **O**bservation **R**eal-Time **I**nterface for **L**ive **I**nformation **E**xchange) is a comprehensive real-time oceanographic data acquisition and processing system developed for research vessels in the US Academic Research Fleet (ARF).

!!! info "Current Status"
    CORIOLIX is actively deployed on Regional Class Research Vessels (RCRVs) and continues to expand its capabilities for the oceanographic research community.

## Background

CORIOLIX was developed during the construction of the Regional Class Research Vessels (RCRVs) for the US Academic Research Fleet (ARF) and established as an ARF Specialized Service Pilot in 2023. The system addresses the critical challenge of ship-to-shore data transmission limitations that have historically kept valuable scientific observations "locked up" onboard until vessels return to port.

## System Capabilities

CORIOLIX provides the following core services:

| Service | Description |
|---------|-------------|
| **Sensor Management** | Configure, monitor, and control onboard instruments |
| **Data Acquisition** | Real-time collection from multiple sensor systems |
| **Data Processing** | Automated processing pipelines with quality control |
| **Local Data Access** | Onboard data storage and retrieval |
| **Remote Data Access** | Shoreside access to vessel data streams |
| **Quality Assurance** | Automated and manual data validation tools |
